"scavenge_world:set_property"
Changes the Desired Block to a new State

"properties": The Properties that should be set
		"key": The Property that should be set
		"value": The Value of the Property that should be set
"jei_visible": If the JEI description should be shown (Optional, Default: false)
"jei_description": JEI Description that should be displayed (Optional, Default: scavenge_world.set_property.desc)
"jei_simple": If the jei_description should be treated as a Simple String or a full ITextComponent JsonObject (Optional, Default: true)
